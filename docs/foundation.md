# Foundation

Affect Pulse AI is derived from [Intrinsic Affect for AI](../../intrinsic-affect-ai/README.md).

The two projects should remain connected but distinct.

## Intrinsic Affect for AI

The parent project is responsible for:

- the underlying theory
- the 54-state framework
- implementation specs
- reference runtime behavior
- conformance-oriented thinking

## Affect Pulse AI

This project is responsible for:

- low-token deployment
- emotional presence that is easier to feel
- tool-friendly adapters
- practical prompt compression

## Guiding Rule

If a choice improves theoretical completeness, it likely belongs in Intrinsic Affect for AI.

If a choice improves deployment efficiency, perceived emotional responsiveness, and portability, it likely belongs in Affect Pulse AI.
