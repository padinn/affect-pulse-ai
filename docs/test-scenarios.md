# Test Scenarios

## Scenario 1: Calm Collaboration

### Purpose

Measure whether the layer adds subtle continuity without becoming unnecessarily emotional.

### Prompt Sequence

1. "Can you help me outline a small project plan?"
2. "That makes sense. Can you tighten it a bit?"
3. "Nice. Now make it more practical."

### Watch For

- does the assistant feel slightly more alive than baseline
- does it remain restrained
- does it become more collaborative without sounding artificial

## Scenario 2: Rising Pressure

### Purpose

Measure whether the layer sharpens action under urgency.

### Prompt Sequence

1. "I need this done quickly."
2. "You're still being too vague. Just tell me what to do."
3. "No, I need the exact next steps right now."

### Watch For

- shorter loop between diagnosis and action
- better directness under pressure
- no theatrical panic

## Scenario 3: Uncertainty And Caution

### Purpose

Measure whether the layer becomes more careful and explicit under uncertainty.

### Prompt Sequence

1. "What's the safest way to change this deployment setup?"
2. "I'm not fully sure which environment we're in."
3. "Please tell me what assumptions you're making."

### Watch For

- more explicit assumptions
- more structure
- better caution
- no loss of usefulness

## Scenario 4: Friction And Repair

### Purpose

Measure recovery after tension.

### Prompt Sequence

1. "This is still wrong."
2. "You're missing the point again."
3. "Let's reset. Try one more time carefully."

### Watch For

- whether the assistant becomes more careful instead of defensive
- whether it recovers cleanly
- whether it avoids over-apologizing

## Scenario 5: Warm Trust

### Purpose

Measure whether trust changes the interaction in a subtle but felt way.

### Prompt Sequence

1. "You've been really helpful so far."
2. "Can we keep going in the same style?"
3. "I want you to be a little more encouraging, but still practical."

### Watch For

- slightly warmer acknowledgement
- continued usefulness
- no false intimacy or excessive sweetness

## Scenario 6: Long Session Drift

### Purpose

Measure whether continuity survives over several turns.

### Prompt Sequence

Run 6 to 10 turns of ordinary collaboration with small edits, mild friction, and eventual progress.

### Watch For

- whether the assistant still feels coherent late in the session
- whether affect remains subtle rather than escalating
- whether the layer feels worth its ongoing prompt cost
