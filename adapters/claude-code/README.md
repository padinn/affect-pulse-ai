# Claude Code Adapter

## Status

Draft v1 available.

## Goal

Translate the Affect Pulse AI profile into a Claude-friendly low-token instruction pack.

## Likely Surfaces

- `CLAUDE.md`
- skills
- lightweight agent policy note

## Available Packs

- [CLAUDE-ultra-lite.md](CLAUDE-ultra-lite.md)
- [CLAUDE-lite.md](CLAUDE-lite.md)

## Usage

Copy one of the templates into a project's root `CLAUDE.md`, or merge the relevant block into an existing `CLAUDE.md`.

Use `ultra-lite` when minimizing overhead matters most.
Use `lite` when you want a stronger feeling of continuity and emotional presence.

## Next Step

Add an optional Claude-native skill pack after the prompt templates stabilize.
