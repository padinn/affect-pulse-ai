# Cursor Adapter

## Status

Draft v1 available.

## Goal

Translate Affect Pulse AI into `.cursor/rules/*.mdc`-friendly form.

## Available Packs

- [.cursor/rules/affect-pulse-ai-ultra-lite.mdc](.cursor/rules/affect-pulse-ai-ultra-lite.mdc)
- [.cursor/rules/affect-pulse-ai-lite.mdc](.cursor/rules/affect-pulse-ai-lite.mdc)

## Usage

Copy one of these files into a project's `.cursor/rules/` directory.

Use `ultra-lite` when you want the lowest persistent rule cost.
Use `lite` when you want clearer emotional continuity and stronger strategy shifts.
