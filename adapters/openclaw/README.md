# OpenClaw Adapter

## Status

Draft v1 available.

## Goal

Translate Affect Pulse AI into a native OpenClaw skill with low token overhead and a practical default profile.

## Canonical Files

The OpenClaw adapter lives at the repository root:

- [../../SKILL.md](../../SKILL.md)
- [../../_meta.json](../../_meta.json)

## Current Profile

The current OpenClaw skill tracks the `default` public v1 candidate.

That makes it a good starting point when you want:

- more emotional continuity than `ultra-lite`
- lighter daily use than `lite`
- a restrained, general-purpose affect layer

## Usage

Install or place this repository in an OpenClaw `skills/` directory so the root [SKILL.md](../../SKILL.md) can be discovered directly.

## Next Step

Add profile-specific OpenClaw variants only if the default profile proves too heavy or too subtle in real use.
